O Projeto Relógio,

É um projeto de desenvolvimento e integração onde através do desenvolvimento de 
diversos relógios.

Visa apresentar técnicas de automação.

São estruturas incorporadas neste projeto:
 - Sensores de temperatura e humidade
 - Relógio com arduino com MP3 e Internet (IoT)
 - Casemod completo para PC.


Veja o link:
http://maurinsoft.com.br/index.php/2020/12/14/nextion-em-pc


O projeto Relógio é um projeto demonstração. 
Apresentando diversos cases com propostas diversas.

Não se trata de um projeto unico, mas um conjunto de integrações entre eletrônica, 
mecânica e informática.
